## Descripción

Intranet desarrollada en React y TypeScript para la gestión de productos, entregas y devoluciones de abastecimiento hospitalario.

El proyecto utiliza Firebase para la autenticación de usuarios y Firestore como base de datos.

## Funcionalidades

- Inicio de sesión de usuarios
- Gestión de productos
- Registro de entregas
- Registro de devoluciones
- Historial de movimientos
- Búsqueda de productos
- Rutas protegidas
- Autenticación con Firebase Auth
- Persistencia de datos con Firestore

## Tecnologías

- React
- TypeScript
- React Router
- Firebase Authentication
- Firebase Firestore
- Vite
